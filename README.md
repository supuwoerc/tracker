# rollup-ts-basic-template

:lollipop: 用于开发前端库的基础模板

## 安装

`git clone https://github.com/supuwoerc/rollup-ts-basic-template.git`

## 启动

### 安装依赖

`pnpm install`

### 启动项目

`pnpm run dev`
